VemTyping

Description : VemTyping est une application web interactive conçue pour aider les utilisateurs à apprendre la dactylographie de manière ludique et progressive.

Fonctionnalités : Leçon avec clavier virtuel, exercices personnalisés, tests de vitesse et progression suivie.

Outils : JavaScript, HTML5, TailwindCSS.

Compétences développées : Programmation front-end, gestion dynamique du DOM, UX design responsive.

Billet de sortie

Description : Application web destinée à numériser et automatiser la gestion des autorisations de sortie pour les étudiants dans un établissement scolaire.

Fonctionnalités : Création de billets de sortie, validation en temps réel, suivi des demandes, notifications.

Outils : ReactJS, CSS, Firebase (backend & base de données).

Compétences développées : Développement full-stack, intégration base de données en temps réel, gestion des états applicatifs.

Blender IA

Description : Projet combinant intelligence artificielle et Blender pour la modélisation et l’animation 3D automatisées.

Fonctionnalités : Modélisation 3D assistée par IA, animation automatique, interface utilisateur web.

Outils : ReactJS, Python, Blender, TensorFlow.

Compétences développées : Programmation IA, intégration de logiciels 3D, développement front-end et back-end.
